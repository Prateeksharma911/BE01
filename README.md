## BE01 and BE03
Todo app in symfony


## API endpoints
* https://127.0.0.1:8000/todoall
* https://127.0.0.1:8000/todo/{id} `Method: POST`
* https://127.0.0.1:8000/todo `Method: PUT`
* https://127.0.0.1:8000/todo/4 `Method: DELETE`

## Features
* CRUD operations
* Symfony Framework
* Connection and use of MYSQL
* Use of Redis Cache
